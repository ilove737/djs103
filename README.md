# DJS-103 Emulator

DJS-103电子计算机模拟器，基于Qt框架实现的古旧计算机仿真程序。

## 项目简介

模拟DJS-103型电子计算机（31位字长，1024字磁鼓内存），支持：

- 汇编代码编辑与加载
- 单步/连续执行
- 寄存器与内存可视化
- LED显示模拟
- 输出控制台

## 构建

```bash
qmake djs103.pro
make
```

## 运行

```bash
./djs103
```

## 操作说明

1. **加载程序**：在代码编辑器中输入汇编代码，点击"装入"按钮
2. **单步执行**：点击"单步"按钮执行一条指令
3. **连续运行**：点击"运行"按钮，支持可调速度
4. **停止/重置**：点击"停止"或"重置"按钮

内置多个示例程序可直接加载测试。

## 技术规格

- 字长：31位（最高位符号位，位1-30为小数部分）
- 内存：1024字磁鼓存储
- 指令格式：两地址指令 [符号][6位操作码XY][12位地址1][12位地址2]
